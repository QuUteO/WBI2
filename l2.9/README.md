# Распаковка строки

## Описание задачи
Функция для примитивной распаковки строки, содержащей повторяющиеся символы/руны.

## Условие
- Написать функцию Go, осуществляющую примитивную распаковку строки, содержащей повторяющиеся символы/руны.
- Функция должна корректно обрабатывать ошибочные случаи (возвращать ошибку через error).
- Поддерживать escape-последовательности вида `\`.
- Код должен проходить проверки (`go vet` и `golint`), т.е. быть написан идиоматически корректно.
- Реализовать проект как модуль Go.

## Примеры работы

### Базовые случаи
| Вход | Выход | Примечание |
|------|-------|-------------|
| `"a4bc2d5e"` | `"aaaabccddddde"` | Обычная распаковка |
| `"abcd"` | `"abcd"` | Нет цифр — без изменений |
| `"45"` | `""` + ошибка | Некорректная строка |
| `""` | `""` | Пустая строка |

### С поддержкой escape-последовательностей
| Вход | Выход | Примечание |
|------|-------|-------------|
| `"qwe\4\5"` | `"qwe45"` | `4` и `5` экранированы |
| `"qwe\45"` | `"qwe44444"` | `\4` экранирует `4`, `5` — число повторений |
| `"qwe\\5"` | `"qwe\5"` | `\\` экранирует обратный слеш |
| `"a\2b\3\4c"` | `"a2b34c"` | Смешанные escape-последовательности |

## Технологии
- **Go**
- **Стандартная библиотека** (без внешних зависимостей)
