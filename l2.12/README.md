# Grep Utility - Аналог UNIX-утилиты grep

## Описание

Реализация упрощённого аналога UNIX-утилиты `grep` для фильтрации текстового потока на языке Go.
Программа читает строки из файла или стандартного ввода (STDIN) и выводит строки,
соответствующие заданному шаблону (подстроке или регулярному выражению).

Поддерживаются различные режимы поиска: контекстный вывод, игнорирование регистра,
инвертированный поиск, нумерация строк, подсчёт совпадений и другие.

## Установка и запуск

```bash
# Клонирование репозитория
cd l2.12

# Сборка бинарного файла
go build -o mygrep

# Запуск тестов
go test -v

# Использование скомпилированного файла
./mygrep [OPTIONS] [PATTERN] [FILE]
