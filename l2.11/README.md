# Find Anagram Sets - Поиск множеств анаграмм

## Описание

Реализация функции для поиска и группировки анаграмм в заданном словаре.
Функция принимает срез слов на русском языке и возвращает map, где ключом
является первое встреченное слово из множества анаграмм, а значением —
отсортированный список всех слов, принадлежащих этому множеству.

Множества, состоящие из одного слова, исключаются из результата.
