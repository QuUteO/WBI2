# WB Tech School — Level 2 (Golang)

Репозиторий содержит решения практических задач и паттернов проектирования для этапа L2 Техношколы Wildberries. Все проекты реализованы на стандартной библиотеке Go с упором на чистую архитектуру и многопоточность. Для запуска любого задания перейдите в его директорию и выполните `go run main.go`, а для проверки кода запустите тесты командой `go test -v -race ./...`.
